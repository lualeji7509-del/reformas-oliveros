# Guía rápida para editar la web

La web está repartida en archivos. Esto es lo que toca cada uno:

- **index.html** — los **textos y el contenido** (servicios, proyectos, FAQ, teléfono, email…). Si quieres cambiar una palabra, un precio o un dato, es aquí.
- **css/estilos.css** — el **diseño**: colores, tamaños, espacios. Si quieres cambiar un color o algo se ve raro, es aquí.
- **js/main.js** — el **comportamiento**: animaciones al hacer scroll, el menú del móvil y el envío del formulario por WhatsApp.
- **imagenes/logo.png** — el **icono** de la pestaña del navegador.

## Datos que están "de ejemplo" (cambiar cuando los tengas)
